IMPLEMENTATION NOTES

从设计表到游戏系统

这套字典最适合转成事件总线 + 目标解析器 + 指令执行器的结构。

合理的部分

Trigger、Scope、Effect、Aura 分层清晰，适合拆成可组合的系统模块；Nested Effect 能承载复杂卡牌，不需要每张卡单独写代码。

需要规范的部分

需要统一 code name、参数括号、区域流转、Buff 持续时间、Aura 叠加规则，以及每类事件携带的 payload 字段。

推荐实装方式

Unity 中用 JSON 或 ScriptableObject 存效果定义，由 Event Bus 发布事件，Scope Resolver 找目标，Effect Executor 通过 Command Log 执行并记录结果。
